# Hermes Grant Scout — Інструкція отримання API ключів

Цей документ містить покрокові інструкції для отримання всіх необхідних API-ключів.

---

## 1. OpenRouter API Key

OpenRouter — єдина платформа для доступу до сотень LLM-моделей (включно з безкоштовними).

**Кроки:**
1. Перейдіть на [openrouter.ai](https://openrouter.ai)
2. Натисніть **Sign In** (правий верхній кут) → зареєструйтесь через Google або GitHub
3. Після входу перейдіть: **Account** → **Keys**
4. Натисніть **Create Key**
5. Дайте назву ключу (наприклад, `grant-scout`)
6. Скопіюйте ключ — він починається з `sk-or-v1-...`

**Налаштування пресету**:
Система налаштована на використання пресету `f-irst`. Створіть його в налаштуваннях OpenRouter ([openrouter.ai/settings/presets](https://openrouter.ai/settings/presets)) та додайте до нього бажані моделі (наприклад, безкоштовні моделі: `google/gemma-3-27b-it:free`).

Переглянути всі безкоштовні моделі: [openrouter.ai/models?max_price=0](https://openrouter.ai/models?max_price=0)

**Записати в `.env`:**
```
OPENROUTER_API_KEY=sk-or-v1-ваш-ключ-тут
```

## 2. Telegram Bot Token

1. Відкрийте Telegram → знайдіть **@BotFather**
2. Створіть нового бота `/newbot` або надішліть `/mybots` → оберіть вашого бота
3. **API Token** → скопіюйте токен

**Ваш Telegram Chat ID:**
1. Знайдіть бота **@userinfobot** у Telegram
2. Надішліть йому будь-яке повідомлення
3. Він поверне ваш числовий ID (наприклад: `123456789`)

**Записати в `.env`:**
```
TELEGRAM_BOT_TOKEN=1234567890:AAH-ваш-токен-тут
TELEGRAM_CHAT_ID=123456789
```

## 3. Notion Integration Token

**Кроки для створення інтеграції:**
1. Перейдіть на сторінку розробника Notion: **[notion.so/developers/connections](https://www.notion.so/developers/connections)**
   *(Сюди також автоматично перенаправляє при спробі перейти на `notion.so/my-integrations` або у вашому Notion-робочому просторі через: ліве меню → **Settings & members** → вкладка **Connections** → посилання **Develop your own connections** внизу сторінки).*
2. Натисніть кнопку **+ New connection**.
3. У модальному вікні, що з'явиться, заповніть такі поля:
   - **Connection name**: `Grant Scout`
   - **Authentication method**: оберіть **Access token**
   - **Installable in**: оберіть ваш робочий простір (workspace) з випадного списку праворуч
4. Натисніть кнопку **Create connection**.
5. Скопіюйте отриманий секретний токен (він починається з `ntn_...` і зазвичай підписаний як **Internal Integration Token** або **Secrets**).

**Підключення сторінки та отримання її ID:**
1. Відкрийте Notion у браузері або додатку.
2. Створіть нову або відкрийте існуючу сторінку, яка буде батьківською (наприклад, з назвою "Гранти").
3. **Важливо:** Надайте інтеграції доступ до цієї сторінки:
   - Натисніть кнопку з трьома крапками `•••` у правому верхньому куті сторінки.
   - Прокрутіть меню вниз до розділу **Connections** (або **Connect to**).
   - Натисніть **Connect to** (або **Add connections**).
   - У пошуку знайдіть створену вами інтеграцію `Grant Scout` та оберіть її.
   - Підтвердіть підключення (натисніть **Confirm**), якщо з'явиться запит.
4. Скопіюйте URL-адресу цієї сторінки з адресного рядка браузера або через меню сторінки (Copy link).
   Приклад URL:
   ```
   https://www.notion.so/my-workspace/Grant-Scout-XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
   ```
5. **Page ID** — це 32-значний буквено-цифровий код наприкінці URL-адреси сторінки (у прикладі вище це `XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX`). Якщо в кінці посилання є параметри запиту на кшталт `?pvs=4`, їх копіювати **не потрібно**.

**Записати в `.env`:**
```
NOTION_API_KEY=ntn_ваш-токен-тут
NOTION_PAGE_ID=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## 4. Web Search Settings (Пошук по вебу)

Grant Scout підтримує два провайдери для пошуку інформації по вебу. Ви можете обрати один з них у змінній `SEARCH_PROVIDER` в `.env`:
* `duckduckgo` — повністю безкоштовний, без API-ключів та лімітів.
* `serper` — Google Search API через Serper.dev (пошук по всьому вебу, 2500 безкоштовних запитів).

---

### 4.1. DuckDuckGo (Безкоштовно та без ключів)
Найшвидший спосіб запуску. Не потребує реєстрації та API-ключів.

**Записати в `.env`:**
```env
SEARCH_PROVIDER=duckduckgo
```

---

### 4.2. Serper.dev (Google API по всьому вебу)
Рекомендований спосіб. Здійснює пошук за допомогою оригінального алгоритму Google по всьому інтернету (без обмежень на сайти). Надає 2 500 безкоштовних пошукових запитів при реєстрації.

**Кроки:**
1. Перейдіть на сайт **[serper.dev](https://serper.dev/)**
2. Зареєструйтесь (через Google або email).
3. Після входу в панель керування (Dashboard) знайдіть поле **API Key** та скопіюйте його.

**Записати в `.env`:**
```env
SEARCH_PROVIDER=serper
SERPER_API_KEY=ваш-ключ-serper-тут
```

## 5. GCP Project ID

1. Відкрийте [console.cloud.google.com](https://console.cloud.google.com)
2. У верхньому рядку натисніть на назву проєкту → відкриється список проєктів
3. Знайдіть ваш проєкт → у колонці **ID** скопіюйте **Project ID**
   (це не назва, а технічний ідентифікатор, наприклад: `my-project-123456`)

**Записати в `.env`:**
```
GCP_PROJECT_ID=your-project-id-123456
```
